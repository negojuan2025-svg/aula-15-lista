# aula-15-lista
# Exercícios de Lógica com Python 🐍

Este repositório contém uma série de exercícios práticos desenvolvidos para exercitar conceitos fundamentais de programação em Python, como manipulação de listas, estruturas de repetição (`while`, `for`), condicionais e conversão de dados.

## 📋 Exercícios Inclusos

### 1. Lista de Convidados
Programa que cadastra 5 convidados em uma lista, exibe todos os nomes e informa o total de registros.

### 2. Controle de Preços
Solicita 5 valores numéricos, armazena-os e identifica automaticamente qual é o **maior** e o **menor** preço informado.

### 3. Lista de Tarefas (To-Do)
Permite a inserção dinâmica de tarefas. O programa só encerra quando o usuário digita a palavra-chave "fim", exibindo a lista completa ao final.

### 4. Controle de Temperaturas
Recebe diversas temperaturas em Celsius e as converte para Fahrenheit utilizando a fórmula:
$F = C \times 1.8 + 32$
Ao final, exibe a média aritmética de ambas as listas de temperatura.

### 5. Sistema de Gestão de Compras
Um sistema completo com menu interativo que permite:
- **Cadastrar** produtos (com parada via comando "sair").
- **Pesquisar** se um item está na lista.
- **Remover** produtos específicos.
- **Alterar** o nome de produtos já cadastrados.
*Obs: Todos os itens são tratados em letras minúsculas para evitar erros de busca.*
